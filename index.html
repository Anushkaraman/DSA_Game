Recursion

Recursion is when a function calls itself to solve smaller instances of a problem. It simplifies solving complex problems by breaking them into simpler subproblems, like in tree traversals or calculating factorials.

Sorting algorithms

Simulations

  • Tower of Hanoi

    Towet of hanoi

    The Tower of Hanoi is a puzzle with three rods and several disks of different sizes. The goal is to move all disks from the first rod to the third, following these rules: only one disk can be moved at a time, larger disks can't be placed on smaller ones, and the second rod is used as a helper. The minimum moves required are (2^n - 1), where n is the number of disks.

  • N-Queen

    N-Queen

    The N-Queen problem involves placing N queens on an N×N chessboard such that no two queens can attack each other. This means no two queens can share the same row, column, or diagonal. The challenge is to find all possible configurations where these conditions are met.

  • BST

    BST Visualization

    A Binary Search Tree (BST) is a tree where each node has up to two children. For each node, values in the left subtree are smaller, and values in the right subtree are larger. This allows for fast search, insert, and delete operations.
